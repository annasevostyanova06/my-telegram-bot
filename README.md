# Telegram Bot Portfolio Project

A Python-based Telegram bot that provides programming advice and quotes.

## Features
- `/start` - Welcome message
- `/advice` - Random programming advice
- `/quote` - Inspirational quotes

## Tech Stack
- Python 3
- python-telegram-bot library
- Deployed on Render.com

## Live Demo
Bot is running 24/7 on Render: @ByteMABuddyBot
